Endpoint Reference Help

REQUEST: /

RESPONSE:

A home page populated with a search bar and search filters.

REQUEST: /help

RESPONSE:

This message.

The following endpoints demonstrate a few of the different combinations of search filters that may be applied to the search.

REQUEST: /api/characters-spells-potions/<search_text>

RESPONSE:

A list of dictionaries from the characters and potions tables that match the user's search string.

REQUEST: /api/spells-potions/<search_text>

RESPONSE:

A list of dictionaries from the spells and potions tables that match the user's search string.

REQUEST: /api/characters/<search_text>

RESPONSE:

A list of dictionaries from the characters table that match the user's search string.

(Others are:

  • /api/characters-spells/<search_text>
  • /api/characters-potions/<search_text>
  • /api/spells/<search_text>
  • /api/potions/<search_text>)
